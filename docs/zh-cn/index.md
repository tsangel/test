# dicomsdl 文档

dicomsdl 是一个轻量级 DICOM 解析器 / 封装库，包含 C++ 核心和基于 nanobind 的 Python 模块。文档优先面向 Python 用户编写，同时也覆盖 C++ API。

英文文档是基准原文。各本地化站点在 `docs/ko`、`docs/ja` 和 `docs/zh-cn` 下共享相同的相对路径。
