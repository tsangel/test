# DicomSDL

DicomSDL 是一个提供 C++ 核心和可选 Python 绑定的轻量级 DICOM 解析器 / 包装库。
这个页面是面向简体中文用户的仓库入口。仓库文档和公开 API 说明会先更新英文版本，中文文档再跟进同步。
