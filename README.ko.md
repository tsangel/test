# DicomSDL

DicomSDL은 C++ 코어와 선택적 Python 바인딩을 제공하는 경량 DICOM 파서/래퍼입니다.
이 페이지는 한국어 사용자용 저장소 진입점입니다. 저장소 문서와 공개 API 설명은 영어를 기준으로 먼저 갱신하고, 한국어 문서는 그 변경을 따라갑니다.
